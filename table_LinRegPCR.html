Application Description

TableProcessingTools - LinRegPCR is a tool to compare LinRegPCR results Pascal with Python versions. It is aware of the correct columns and ignores small float differences.

Accepted Input

The table files can be provided in txt, tsv or csv format (*.txt, *.tsv, and *.csv).

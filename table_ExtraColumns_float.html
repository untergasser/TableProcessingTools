Application Description

TableProcessingTools - ExtraColumns is a tool to compare identical columns in two table files. The identical columns are identified by the same header id in the first column.

Accepted Input

The table files can be provided in txt, tsv or csv format (*.txt, *.tsv, and *.csv).
